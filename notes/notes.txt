what mmap is doing?

In modern x86-64 architectures, the transition from user space to kernel space during a system call is highly optimized using Model-Specific Registers (MSRs). 
When a process executes the syscall instruction, the CPU performs a rapid context switch by reading specific MSRs previously initialized by the kernel at boot time. 
The most critical of these is the MSR_LSTAR (Long System Target Address Register), which holds the exact memory address of the kernel's system call entry routine (typically entry_SYSCALL_64).
